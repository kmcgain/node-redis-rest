node-redis-rest
===============
## Usage

 Clone the repo and install dependencies:

    $ git clone https://github.com/nickholub/node-redis-rest
    $ cd node-redis-rest
    $ npm install

 Run Redis:

    $ <redis>/src/redis-server

 Start the server:

    $ node app

  Test REST API, e.g.

    http://localhost:3000/keys/*
    http://localhost:3000/key/foo
